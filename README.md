# HRMS Lite

A simple Human Resource Management System built with React and FastAPI.

## Project Overview

HRMS Lite is a web application for managing employees and tracking attendance. The system provides a clean interface to add/view/delete employees and mark attendance records with basic statistics.

## Tech Stack

**Frontend:**
- React 18 with Vite
- React Router v6
- Axios for API calls
- CSS3 (responsive design)

**Backend:**
- FastAPI
- SQLAlchemy ORM
- PostgreSQL / SQLite

## Running Locally

### Prerequisites
- Node.js 16+
- Python 3.8+
- pip

### Frontend Setup

```bash
cd hrms-frontend
npm install
npm run dev
```

The frontend will run on `http://localhost:5173`

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
python -m uvicorn app.main:app --reload
```
